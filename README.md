# Hangman Game (Java)

This is a Java Swing based Hangman game with a graphical user interface.

## Features
- GUI using Java Swing
- Visual hangman drawing for wrong guesses
- Simple and interactive gameplay

## Technologies Used
- Java
- Java Swing (GUI)

## How to Run
1. Open terminal in the project folder
2. Compile the program:
   javac HangmanGUI.java
3. Run the program:
   java HangmanGUI

## File Structure
- HangmanGUI.java – Main game source code
- README.md – Project documentation
